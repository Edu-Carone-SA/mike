01
Introdução ao Mike
Governança simples, acessível e digital

O Mike é um assistente jurídico com inteligência artificial que ajuda profissionais de governança e direito a analisar documentos, responder questões jurídicas, redigir textos, realizar revisões tabulares e pesquisar jurisprudência — tudo em um só lugar.

O que o Mike faz

💬
Assistant
Converse com a IA, anexe documentos, gere arquivos e pesquise jurisprudência.
📁
Projects
Organize documentos, chats e revisões por projeto. Compartilhe com o time.
📊
Tabular Review
Extração estruturada de dados de múltiplos documentos em formato de grade.
⚙️
Workflows
Modelos reutilizáveis de prompt e configurações de colunas para automação.

Primeiros passos

  1. Faça login
    Acesse https://mike.agov.app e entre com seu e-mail e senha. Se a autenticação multifator (MFA) estiver ativada, você precisará informar o código de 6 dígitos do seu app autenticador.
  2. Navegue pela barra lateral
    A barra lateral à esquerda dá acesso às quatro principais funcionalidades: Assistant, Projects, Tabular Review e Workflows. Também mostra projetos recentes e o histórico de chats.
  3. Configure sua conta
    Clique no seu nome na barra lateral para acessar Configurações da Conta, onde você pode ajustar perfil, segurança, chaves de API, conectores e preferências de modelo.
  4. Comece a usar
    A maneira mais rápida de começar é abrir o Assistant e fazer uma pergunta. Você pode anexar documentos, aplicar workflows e gerar arquivos — tudo a partir do chat.
📌 Sobre roles de usuário

O Mike possui dois perfis: admin e member. Administradores podem gerenciar usuários, alterar roles, resetar senhas e visualizar logs de auditoria. Membros usam o sistema normalmente. A criação de usuários é feita apenas por administradores — não há auto-registro.

02
Assistant
Chat com IA jurídica · Análise de documentos · Geração de arquivos

O Assistant é o coração do Mike. É uma interface de chat onde você conversa com uma IA jurídica que pode ler documentos anexados, buscar informações dentro deles, gerar arquivos (Word, Excel, PowerPoint), propor edições com controle de alterações e pesquisar jurisprudência americana via CourtListener.

Como usar o Assistant

  1. Abra o Assistant
    Clique em "Assistant" na barra lateral. Se não houver chats anteriores, você verá uma tela de boas-vindas com um campo de mensagem.
  2. Escreva sua mensagem
    Digite sua pergunta ou instrução no campo de texto. Você pode ser específico ("Revise a cláusula de confidencialidade do documento anexado") ou abrangente ("Faça um resumo dos riscos principais deste contrato").
  3. Anexe documentos (opcional)
    Clique no botão "Add Documents" para anexar arquivos. Formatos suportados: PDF DOCX DOC XLSX XLSM XLS PPTX PPT. Documentos Word e PowerPoint são convertidos para PDF automaticamente para visualização.
  4. Aplique um workflow (opcional)
    Clique no botão "Workflows" para selecionar um workflow (modelo de prompt) que guiará a resposta da IA. Veja a seção 05 para detalhes.
  5. Envie a mensagem
    Pressione Enter ou clique no botão de envio. A resposta da IA aparece em streaming, token por token. O chat é salvo automaticamente no histórico.

Recursos do Assistant

📄 Geração de arquivos

A IA pode gerar documentos completos sob demanda:

  • Word (.docx): documentos com seções, tabelas, cabeçalhos e quebras de página.
  • Excel (.xlsx): planilhas com múltiplas abas.
  • PowerPoint (.pptx): apresentações com slides estruturados.

Os arquivos gerados ficam disponíveis para download direto no chat.

✏️ Edição com controle de alterações

Quando você anexa um documento .docx, a IA pode propor edições precisas (substituições de texto). Cada edição aparece como um cartão de Aceitar / Rejeitar. Ao aceitar, uma nova versão do documento é criada com a alteração aplicada.

💡 Dica

A IA renumera cláusulas e atualiza referências cruzadas automaticamente ao propor edições. Revise sempre — a IA é precisa, mas o contexto jurídico exige validação humana.

📑 Citações

Quando a IA responde com base em um documento anexado, ela inclui citações numeradas [1], [2], etc. Clique em uma citação para abrir a passagem original no painel lateral, com o trecho destacado.

⚖️ Pesquisa de jurisprudência (CourtListener)

Se a pesquisa jurisprudencial estiver ativada nas configurações da conta, a IA pode:

  • Verificar citações de casos jurídicos;
  • Buscar casos por termo ou filtros;
  • Ler o conteúdo de casos específicos;
  • Pesquisar na base de jurisprudência americana.

Os resultados aparecem em um painel dedicado com metadados do caso, opiniões e links para o PDF original.

🔄 Chat com escopo de projeto

Você pode iniciar um chat dentro de um projeto específico. Nesse caso, o assistente tem acesso aos documentos do projeto automaticamente. Acesse via Projects → [projeto] → Assistant → New Chat.

🤖 Modelos de IA

O Mike utiliza modelos da família DeepSeek. No seletor de modelo (canto superior do chat), você pode escolher entre:

Modelo Uso Característica
DeepSeek V4 Pro Chat principal Mais capaz e detalhado
DeepSeek V4 Flash Títulos, extração tabular Mais rápido e econômico
⚠️ Atenção

A IA pode usar até 10 rodadas de ferramentas por resposta (leitura de documentos, buscas, geração de arquivos). Se a resposta parecer incompleta, refine a pergunta ou divida em etapas.

03
Projects
Organize documentos, chats e revisões por projeto

Projects é o espaço de trabalho para organizar documentos, chats do assistant e revisões tabulares em um mesmo contexto. Ideal para gerenciar um caso, transação ou due diligence. Projetos podem ser compartilhados com outros usuários.

Criando um projeto

  1. Acesse a lista de projetos
    Clique em "Projects" na barra lateral. Você verá a lista de projetos com filtros: All, Mine, Shared with me.
  2. Clique em "New project"
    Preencha: nome do projeto, CM number (número do caso/matéria), practice (área de prática) e, opcionalmente, e-mails de usuários para compartilhar.
  3. Confirme
    O projeto é criado e você é levado ao workspace do projeto.

Workspace do projeto

Dentro de um projeto, você tem três seções em abas:

📄
Documents
Upload, organização em pastas, versionamento e download de documentos. Arraste e solte ou use o seletor. Cada documento pode ter múltiplas versões.
💬
Assistant
Chats do assistant com escopo no projeto. A IA tem acesso automático aos documentos do projeto. Clique em "New Chat" para iniciar.
📊
Tabular Reviews
Revisões tabulares vinculadas ao projeto. Clique em "New Review" para criar uma extração estruturada a partir dos documentos do projeto.

Gerenciando documentos

Na seção Documents do projeto, você pode:

  • Upload: arraste arquivos ou use o seletor. Formatos: PDF, DOCX, DOC, XLSX, XLSM, XLS, PPTX, PPT.
  • Pastas: crie pastas e subpastas para organizar documentos.
  • Versionamento: suba novas versões de um documento, substitua o arquivo, copie versão de outro documento, ou exclua versões antigas.
  • Download: baixe documentos individuais ou um ZIP de toda a pasta.
  • Mover: mova documentos entre pastas.
  • Renomear: altere o nome do documento.

Compartilhamento

O dono do projeto pode compartilhá-lo com outros usuários do Mike por e-mail. Usuários compartilhados têm acesso de leitura e escrita, mas apenas o dono pode:

  • Editar detalhes do projeto;
  • Excluir o projeto;
  • Gerenciar compartilhamento.
📌 Exclusão de projeto

A exclusão de um projeto remove permanentemente todos os documentos, versões, arquivos armazenados, chats e revisões tabulares associados. Esta ação não pode ser desfeita.

04
Tabular Review
Extração estruturada de dados de múltiplos documentos

Tabular Review é uma ferramenta de due diligence em grade: cada linha é um documento, cada coluna é um campo de extração. A IA preenche cada célula analisando o documento correspondente, com citação da fonte. Ideal para extrair informações estruturadas de contratos, acordos e documentos em volume.

Criando uma revisão tabular

  1. Acesse Tabular Review
    Clique em "Tabular Review" na barra lateral, ou abra um projeto e vá para a aba Tabular Reviews.
  2. Clique em "New Tabular Review"
    Dê um título, selecione os documentos que farão parte da revisão e, opcionalmente, aplique um workflow tabular para preencher as colunas automaticamente.
  3. Configure as colunas
    Defina quais informações você quer extrair de cada documento. Cada coluna tem um nome, um formato e um prompt que orienta a IA.
  4. Gere os dados
    Clique em "Generate". A IA processa cada documento e preenche as células em streaming. Você pode regenerar células individuais depois.

Formatos de coluna

Cada coluna pode ter um formato específico que define como a IA deve estruturar a resposta:

Formato Descrição Exemplo
Text Texto livre Resumo da cláusula
Bulleted List Lista de pontos • Item 1 • Item 2
Number Valor numérico 42
Percentage Percentual 15%
Monetary Amount Valor monetário R$ 50.000
Currency Moeda BRL
Yes/No Sim ou Não Yes
Date Data 2026-07-12
Tag Etiqueta Alta prioridade

Trabalhando com a grade

Estados das células

  • Pending Aguardando geração
  • Generating IA processando
  • Done Extração concluída
  • Error Falha na extração

Painel lateral

Clique em qualquer célula para abrir o painel lateral, que mostra o conteúdo extraído, a citação da fonte (trecho do documento) e o raciocínio da IA.

Chat da revisão

Cada revisão tem um chat dedicado onde você pode fazer perguntas sobre os dados extraídos. A IA pode ler o conteúdo das células para responder.

Exportação

📤 Exportar para Excel

Clique em "Export Excel" para baixar a revisão completa como um arquivo .xlsx, com todas as colunas, documentos e dados extraídos.

💡 Dica de economia

A geração tabular usa o modelo tabular (configurável em Configurações → Models). Recomendamos usar um modelo menor e mais econômico (como o DeepSeek V4 Flash) para reduzir custos em revisões com muitos documentos.

05
Workflows
Modelos reutilizáveis de prompt e configuração de colunas

Workflows são modelos reutilizáveis que automatizam tarefas recorrentes. Existem dois tipos:

💬
Assistant Workflow
Um prompt em texto rico que orienta como a IA deve analisar documentos e responder. Exemplo: "NDA Review" instrui a IA a verificar cláusulas específicas de um acordo de confidencialidade.
📊
Tabular Workflow
Um conjunto de colunas pré-definidas que pode ser aplicado a uma revisão tabular. Exemplo: "SPA" define colunas para extrair dados de um Acordo de Compra e Venda de Ações.

Criando um workflow personalizado

  1. Acesse Workflows
    Clique em "Workflows" na barra lateral.
  2. Clique em "New workflow"
    Defina: título, tipo (Assistant ou Tabular), idioma, área de prática e jurisdição.
  3. Edite o conteúdo
    Para workflows do tipo Assistant, escreva o prompt em texto rico no editor. Para workflows do tipo Tabular, defina as colunas (nome, formato, prompt, tags). O salvamento é automático.
  4. Use ou compartilhe
    Clique em "Use" para aplicar o workflow em um novo chat ou revisão. Clique em "Share" para compartilhar com outros usuários (com permissão de visualização ou edição).

Usando um workflow

Você pode aplicar um workflow de duas formas:

  • No Assistant: ao escrever uma mensagem, clique em "Workflows" e selecione um workflow. A IA lerá as instruções do workflow e seguirá o prompt ao responder.
  • No Tabular Review: ao criar uma nova revisão, selecione um workflow tabular para preencher as colunas automaticamente. Ou, numa revisão existente, clique em "Apply Workflow".
💡 Dica

Workflows podem ser exportados como arquivo ZIP para backup ou compartilhamento externo. Use o botão "Download" na página de detalhes do workflow.

06
Conta e Configurações
Perfil · Segurança · Chaves de API · Conectores · Administração

Acesse as configurações da conta clicando no seu nome na barra lateral. As configurações estão organizadas em abas:

Perfil (General)

  • Nome de exibição e organização — editáveis.
  • E-mail — pode ser alterado (requer MFA se ativado).
  • Excluir conta — remove permanentemente a conta e todos os dados (requer MFA).

Features

  • Pesquisa Jurídica (CourtListener): ativa a pesquisa de jurisprudência americana no assistant.

Privacidade e Dados

  • Exportar dados: baixe chats (JSON), revisões tabulares (JSON) ou conta completa.
  • Excluir dados: remova todos os chats, revisões ou projetos (requer MFA).

Segurança

  • Autenticação Multifator (MFA): ative com um app autenticador (TOTP). Escaneie o QR code e verifique com um código de 6 dígitos.
  • MFA no login: force a verificação MFA a cada login (se um fator verificado existir).

Preferências de Modelo

  • Modelo de títulos: usado para gerar títulos de chat e extrações leves.
  • Modelo tabular: usado na geração de células de revisão tabular.

Chaves de API

  • DeepSeek API Key: adicione sua chave pessoal. Se a chave for gerenciada pelo servidor (Atlas), aparece como "Configured by Atlas".
  • CourtListener API Key: opcional, para dados de jurisprudência ao vivo.

Conectores (MCP)

Adicione servidores de ferramentas externas via Model Context Protocol (MCP). Configure por URL com token Bearer e headers personalizados. Suporta OAuth para servidores que exigem autorização. Habilite/desabilite conectores e ferramentas individuais.

Administração de Usuários admin

Disponível apenas para administradores. Permite:

  • Listar usuários: e-mail, role, status, data de criação, último login.
  • Criar usuário: e-mail + role (member/admin). Gera senha temporária (mostrada uma vez).
  • Alterar role: promover/demover entre admin e member.
  • Desabilitar/Habilitar: bloqueia o acesso do usuário.
  • Resetar senha: gera nova senha temporária.
  • Revogar sessões: força re-login.
⚠️ Proteções do sistema

O sistema impede: auto-desabilitamento, remoção do último administrador ativo e acesso de membros à área de administração (403). Todas as ações administrativas são registradas no log de auditoria.
